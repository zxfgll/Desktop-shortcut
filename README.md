## 📖 常见问题（FAQ）

**Q1: 如何下载安装？**  
A1: 您可以直接在 [App Store](https://apps.apple.com) 下载最新版本。  

**Q2: 我付费后在另一台设备上能恢复吗？**  
A2: 可以，使用同一个 Apple ID 登录后，在应用中点击「恢复购买」即可。  

**Q3: 应用崩溃 / 无法启动怎么办？**  
A3: 请先尝试重启设备，如果问题仍然存在，请联系我们的支持团队。
